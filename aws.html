What We Cover

- EC2: Virtual servers in the cloud.
- Lambda: Serverless computing for running code.
- S3: Object storage for data and files.
- EFS: Scalable file storage for EC2.
- VPC: Create secure cloud networks.
- CloudFront: Fast content delivery across the globe.
- ELB: Distribute traffic to multiple servers.
- ASG: Automatically scale your application.
- SNS: Send notifications to users.
- SQS: Message queuing for decoupling services.
- RDS: Managed relational databases.
- DynamoDB: Fast NoSQL database service.
- CloudWatch: Monitor resources and applications.
- IAM: Manage user access and permissions.
- CloudFormation: Manage AWS resources with code.
- EventBridge: Integrate applications using events.
- Migration & Snowball: Tools for moving data to AWS.
- Elastic Beanstalk: Easy app deployment and management.
- Mini Project: Web hosting using EC2.
- Main Project: Code deployment and pipeline setup.
